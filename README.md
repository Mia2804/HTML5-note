# HTML5-note
HTML是超文本，不是编程语言
# 1、基础标签
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
    Mia
</body>
</html>

## 声明
<!DOCTYPE html>声明代表html5版本！ 
## <html>
<html>
</html>
所有数据都要包含在其中
  
## head
<head></head>
定义基础信息，编码格式如UTF-8，标题，文字显示形式（如en代表英文）
<title></title>
  
# 2.标签（标签的使用是成对出现的）
HTML元素指的是从开始标签到结束标签所有的代码
    <h1>title_h1</h1>
    <h2>title_h1</h2>
    <h3>title_h1</h3>
    <h4>title_h1</h4>
    <h5>title_h1</h5>
    <h6>title_h1</h6>
    代表了从大到小的六个字号title
